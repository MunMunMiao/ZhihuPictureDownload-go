# ZhihuPictureDownload

## How to use

#### Download
[Releases](https://github.com/MunMunMiao/ZhihuPictureDownload-go/releases)

#### Quick start
```text
./zhihu -u https://www.zhihu.com/people/tserendipity
```

## Options
```text
Usage: zhihu [options]

Options:
  -d Save images
  -u User home page
```
